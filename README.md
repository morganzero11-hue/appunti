# Appunti

Un sito semplice per pubblicare e condividere appunti.

## Come usare

1. Apri `index.html` nel tuo browser per visualizzare il sito.
2. Modifica il contenuto delle sezioni in `index.html` per aggiungere i tuoi appunti.
3. Personalizza lo stile in `style.css` se necessario.

## Struttura

- `index.html`: La pagina principale con gli appunti.
- `style.css`: Stili CSS per il design.
- `README.md`: Questo file di documentazione.

## Aggiungere nuove note

Per aggiungere una nuova nota:
1. Aggiungi un link nel `<nav>` per la nuova sezione.
2. Crea una nuova `<section>` nel `<main>` con un id corrispondente.
3. Scrivi il contenuto della nota nella sezione.
